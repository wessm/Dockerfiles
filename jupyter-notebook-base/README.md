# jupyter-notebook-base

based on [jupyter/scipy-notebook](https://hub.docker.com/r/jupyter/scipy-notebook/)

contains:
* postgis
* imagemagick
* gdal binaries
* exiftool
* fortran
* gcc

python modules:
* geopandas
* shapely
* psycopg2
* pyshp
* imutils
* cython
* pyexiftool
* fortran-magic
* autopep8
* yapf
